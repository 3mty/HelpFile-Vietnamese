###User Defined Function###
_GUICtrlIpAddress_SetFont

###Description###
Set font of the control

###Syntax###
#include <GuiIPAddress.au3>
_GUICtrlIpAddress_SetFont ( $hWnd [, $sFaceName = "Arial" [, $iFontSize = 12 [, $iFontWeight = 400 [, $bFontItalic = False]]]] )

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$sFaceName
	[optional] The Font Name of the font to use
$iFontSize
	[optional] The Font Size
$iFontWeight
	[optional] The Font Weight
$bFontItalic
	[optional] Use Italic Attribute
@@End@@

###ReturnValue###
None.

###Remarks###
None.

###User Defined Function###
_GUICtrlListView_GetItemSpacingX

###Description###
Determines the horizontal spacing between items in the control

###Syntax###
#include <GuiListView.au3>
_GUICtrlListView_GetItemSpacingX ( $hWnd [, $bSmall = False] )

###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$bSmall
	[optional] View for which to retrieve the item spacing:
		True - Small icon view
		False - Icon view
@@End@@

###ReturnValue###
Returns the horizontal spacing.

###Remarks###
None.

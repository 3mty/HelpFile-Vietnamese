###User Defined Function###
_GUICtrlToolbar_SetStyleWrapable

###Description###
Sets whether the control is wrapable

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_SetStyleWrapable ( $hWnd, $bState )

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$bState
	True to set, false to unset
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@

###Remarks###
None.

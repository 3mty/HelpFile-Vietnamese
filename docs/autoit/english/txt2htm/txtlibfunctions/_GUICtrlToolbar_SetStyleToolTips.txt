###User Defined Function###
_GUICtrlToolbar_SetStyleToolTips

###Description###
Sets whether the control has tooltips

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_SetStyleToolTips ( $hWnd, $bState )

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$bState
	True to set, false to unset
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@

###Remarks###
None.

###User Defined Function###
_Date_Time_SystemTimeToFileTime

###Description###
Converts a system time to file time format

###Syntax###
#include <Date.au3>
_Date_Time_SystemTimeToFileTime ( $tSYSTEMTIME )

###Parameters###
@@ParamTable@@
$tSYSTEMTIME
	a $tagSYSTEMTIME structure to be converted or a pointer to it.
@@End@@

###ReturnValue###
Returns a $tagFILETIME structure that contains the converted file time.

###Remarks###
None.

###Related###
$tagFILETIME, $tagSYSTEMTIME

###Function###
Abs

###Description###
Calculates the absolute value of a number.

###Syntax###
Abs ( expression )

###Parameters###
@@ParamTable@@
expression
	Any valid numeric expression.
@@End@@

###ReturnValue###
Returns absolute value of <em>expression</em>.

###Remarks###
A string has a value of zero.

None.
